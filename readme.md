# Рабочий каскад в 3x-UI через REALITY

Данное руководство показывает, как настроить реально рабочий каскад
между двумя серверами с помощью 3x-UI

## Схема работы

Клиент → Сервер A → Сервер B → Интернет

------------------------------------------------------------------------

## Шаг 1 --- Установка 3x-UI на оба сервера

```
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```
или 

```
wget https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh
```
```
chmod +x install.sh
```
```
sudo ./install.sh
```
------------------------------------------------------------------------

## Шаг 2 --- Настройка сервера B

Создайте inbound с такими настройками:

  Name - real(может быть любым)
  
  Port - 443
  
  Protocol - VLESS
  
  Security - Reality
  
  target - ads.x5.ru (может быть любым, но одинаковым с сервером А)
  
  SNI - ads.x5.ru (может быть любым,но одинаковым с сервером А)

------------------------------------------------------------------------

## Шаг 3 --- Настройка сервера A

Создайте inbound с такими же параметрами.

Скопируйте inbound с сервера B

И вставьте его в: `Настройка xray → outbound → Add`

Имя outbound будет: `real-xxxxxxxx`

------------------------------------------------------------------------

## Шаг 4 --- Маршрутизация

Network - `TCP,UDP`

outbound tag - `real-xxxxxx`

------------------------------------------------------------------------

## Проверка

Не забывайте каждое изменение сохранять!

Подключитесь к серверу A --- внешний IP должен быть сервера B.
